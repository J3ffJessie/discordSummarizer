# Changelog

## [1.1.0] - 2026-05-01

### Added
- Giveaway Spinner. Embeds URL for initiator to pull up and add prizes if multiple and select which prize is up for grabs. Adds a viewable wheel spin URL for entrants. Simple Enter Giveaway button that automatically captures users username from Discord and adds to the entrants on the wheel. Single entry initially, users can re-enter for multiple prizes if host so chooses. 
- Tracking of Giveaway winners. On each servers dashboard there is a new section for Giveaway winners. This will track who won, what they won and the date they won. No need to keep track on sticky notes. This can be cleared at any time by the admin if they choose to.
- Giveaway winners are sent a DM indicating they have won and to reach out to the Giveaway Host (individual who started the giveaway for further details).
- Configuration to send Coffee Pairings into a specific text channel. Users had experienced issues with Discord not showing them the name of their match but the users ID number instead making it difficult for some users to figure out who they are matched to. Sending to a text channel works around that issue and channel can be controlled so that only those with the coffee pairing role can see the channel if so chosen.
