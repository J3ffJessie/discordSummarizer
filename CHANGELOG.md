# Changelog

## [1.0.0] - 2026-03-30

### Added
- Initial release
- `/summarize` command — summarize recent channel messages
- `/server` command — gather and summarize all channel conversations
- `/translate start|stop` — live voice-to-text translation with captions page
- `/paircoffee` — manual coffee chat pairing (admin only)
- `/remindme` — set personal reminders
- Pluggable AI provider system (Groq, OpenAI, Anthropic, Ollama, custom)
- Release notifications on bot restart
